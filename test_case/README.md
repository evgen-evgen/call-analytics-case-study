# ASR evaluation

Аудиофайлы лежат в `../test_data`, а дословные эталонные расшифровки —
в `references/`. Пары сопоставляются по двузначному номеру:

```text
../test_data/01_credit_16k.wav
references/call_01.txt
```

Допустимы префиксы `О:` и `К:` — evaluator удаляет их перед сравнением.
Оболочка `text = """..."""` также поддерживается. Регистр, пунктуация и
`е/ё` нормализуются. Аудиофайлы без соответствующего эталона пропускаются.

Запуск из корня проекта:

```bash
uv run python -m scripts.evaluate_wer
```

Распознанные тексты сохраняются в `hypotheses/`, таблицы для отчёта — в
`reports/wer.csv` и `reports/wer.md`.
